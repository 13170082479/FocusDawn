# makedawn UI 重构方案

## 1. UI 重构方案

本次重构将 `makedawn - 创作守护` 从传统 tkinter 工具界面迁移到 CustomTkinter。功能逻辑保持不变，重点重做视觉设计、信息层级和交互体验。

设计方向：

- 首页只强调一个核心问题：今天还差多少创作时间。
- 次级指标只保留连续达标、本周达标率、今日娱乐时长。
- 数据分析页改为深色趋势图和热力图。
- 设置页保留必要控制项，减少工具感。
- 日志页改为深色列表，弱化粗边框。

## 2. 页面布局示意图

```text
makedawn
先创作，后娱乐                         状态提示

[ 今日 ] [ 数据分析 ] [ 设置 ] [ 日志 ]

今日页：
+-------------------------------+-----------------------+
| 今日创作进度                  | 当前状态              |
| 进度环                         | 娱乐锁定 / 已解锁     |
| 0 / 60 分钟                    | 状态说明              |
| 进度条                         | 小贴士                |
+-------------------------------+-----------------------+
| 连续达标 | 本周达标率 | 今日娱乐时长          |
+-------------------------------------------------------+
| 开始创作 | 暂停 | 结束 | 刷新                    |
+-------------------------------------------------------+
| 今日创作记录                  | 本周创作概览          |
+-------------------------------------------------------+
```

## 3. CustomTkinter 组件选型

- 主窗口：`ctk.CTk`
- 页面容器：`ctk.CTkScrollableFrame`
- 卡片：`ctk.CTkFrame`
- 标签：`ctk.CTkLabel`
- 按钮：`ctk.CTkButton`
- 输入框：`ctk.CTkEntry`
- 文本框：`ctk.CTkTextbox`
- 开关：`ctk.CTkSwitch`
- 复选框：`ctk.CTkCheckBox`
- 趋势图和进度环：`tk.Canvas`
- 表格：由 `CTkFrame + CTkLabel` 自绘，避免默认 Treeview 风格

## 5. 统一设计规范

- 页面边距：32px
- 卡片间距：24px
- 卡片圆角：12px
- 按钮圆角：10px
- 按钮高度：44px
- 深色模式优先
- 减少粗边框，用背景层级和轻边框区分区域
- 页面导航使用自定义按钮，不使用默认 Notebook
- 数据表格使用自绘列表，不使用默认 Treeview

## 6. 可替换代码实现方案

保留不变：

- `analyzer.py`：统计分析
